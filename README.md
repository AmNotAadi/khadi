# Khadi Skin Care - Organic E-commerce (WhatsApp Checkout)

Quick start:

1. Install Node.js 18+.
2. Install deps: `npm install`
3. Run dev: `npm run dev`
4. Open `http://localhost:3000`

Features:
- Responsive homepage (hero, featured, categories, testimonials, contact)
- Shop with categories and add-to-cart
- Cart and checkout via WhatsApp pre-filled message
- Minimal backend saves orders to `orders.json`

Change WhatsApp number:
- Update `businessNumber` in `public/cart.html` script.

Update stall/event:
- Edit the Stall section in `public/index.html`.
